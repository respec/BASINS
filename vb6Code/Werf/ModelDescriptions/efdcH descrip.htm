General Description:

The EFDC model was originally developed as a three-dimensional estuarine and coastal ocean circulation model and has been subsequently extended to model a wide range of surface water flows, including catchment scale overland flow, flow in vegetated fresh and brackish water wetlands, flow in rivers and channel networks and flow in lakes and reservoirs. The EFDC model is also configurable as a mesoscale, limited-area atmospheric model. The model's general three-dimensional formulation readily reduces to one- and two-dimensions in the vertical or horizontal axes to resolve site-specific problems. The model's hydrodynamic simulation is based on a three-time-level, semi-implicit solution of the vertically hydrostatic equation on a curvilinear-orthogonal horizontal grid with a sigma stretched (or topography following) vertical coordinate. For variable density flows, transport equations for salinity and temperature are also solved. The Mellor-Yamada level 2.5 turbulence closure model is implemented for three- and two-dimensional vertical applications. For modeling flows with intermittent drying and wetting, a robust drying and wetting scheme that prohibits negative depths is implemented. For submerged and emergent vegetated environments, the code includes both three-dimensional and two-dimensional vegetation resistance formulations, with the three-dimensional formulation fully coupled to the turbulence closure model. The model also has provisions for hydraulic control structures such as spillways, weirs, gates and culverts.

The model has the internal capability to simulate the transport and transformation of an arbitrary number of dissolved and suspended constituents using a flux-corrected, positive-definite transport formulation. Transformation kinetics are specified by a user-specified subroutine. Two kinetic subroutines, equivalent to the 8 state variable WASP5 eutrophication model and the 20 state variable CE-QUAL-ICM, are currently being implemented directly into the code to provide a capability for simultaneous hydrodynamic and water quality simulation without the necessity of custom interfacing. The code also has the capability of writing information in the output DYNHYD format transport to drive the WASP5 model in a stand-alone mode. An alternate transport output is implemented to allow the EFDC code to run in a transport and transformation mode. Additional enhancements useful for water quality studies include a Lagrangian plume model for near-field point source loading and interfacing. Two Lagrangian particle tracking schemes are implemented for oil spill simulation and larvae recruitment and transport studies. Planned enhancements to the code include a coupled in-place, three-dimensional groundwater flow model and a sediment-water phase toxics transport model.

Model methods include the following:

A finite difference scheme is used in EFDC with three time levels and an internal-external mode splitting procedure that achieves separation of the internal shear or baroclinic model from the external free-surface gravity wave (barotropic mode). EFDC uses an implicit external mode with simultaneous computation of a two-dimensional surface elevation field by a multicolor successive overrelaxation procedure. Using the new surface elevation field, the external solution is completed by calculation of the depth-integrated barotropic velocities. The advective transport options that can be used include the "centered in time and space" scheme, and the "forward in time and upwind in space" scheme.

Limitations

Development and calibration of the model requires significant technical expertise.

Application History

EFDC has been used for modeling estuaries of the Chesapeake Bay, the Indian River Lagoon in Florida, Lake Okeechobee in Florida , Peconic Bay in New York, Stephens Passage in Alaska, and Nan Wan Bay, Taiwan. In the Florida Everglades, EFDC has been used to simulate large scale wetlands flow and transport.

Model Use Features

The model is written in universal FORTRAN. The model can be run on supercomputers/workstations as well as PCs.
