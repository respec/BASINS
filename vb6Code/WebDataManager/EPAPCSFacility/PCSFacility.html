Selection of Columns for Query Engine



STEP 3: Selection of Columns

Select one or more column(s) for your output by clicking on the square box next to the column name.

TABLE NAME: PCS_PERMIT_FACILITY
NPDES An number formatted to define the scope of a general permit, the industrial classification, and a sequence number.
Count Function for NPDES. DO NOT SELECT THE COLUMN LOCATED DIRECTLY ABOVE I F YOU SELECT THIS COLUMN You may to elect to count the number of distinct records for the column above:(You must select an additional search-by column in addition to this group functions) HELP on STATISTICS!
SIC Code The four-digit code for the principal facility discharging activity, as defined by the 1987 Standard Industrial Classification (SIC) Manual.
Code Expansion for SIC Code Selecting this option will provide you with the description for the code from the field above this one
City Name The city name of the facility location, defined with city code.
County Name The name of the county in which the facility is located.
Region A two-digit code used to identify the EPA region in which the facility is located.
Inactive Date Date the facility became inactive or active.
Application Received Date The date on which the application for a NPDES permit was received.
City Code A unique, five-digit code maintained by EPA for each city and place in a particular state or territory.
Code of Accuracy Code describing technical accuracy of latitude and longitude data.
Code Expansion for Code of Accuracy Selecting this option will provide you with the description for the code from the field above this one
Official Representative The name and/or department of the permittee's representative for Discharge Monitoring Reports (DMR), also referred to as Facility Contact Person.
Official Representative Telephone Number The telephone number of the permittee representative for DMRs.
County Code The three-digit county Federal Information Processing Standards (FIPS) code for the facility location.
Current Year Automatic QNCR Status Indicates the status of reportable non-compliance, as it appeared on the quarterly non-compliance report (QNCR) for the current year.
Current Year Manual QNCR Status Indicates the status of reportable noncompliance for major and minor facilities, according to the quarterly noncompliance report for the current year.
Datum Describes the datum used to determine the latitude and longitude coordinates.
Code Expansion for Datum Selecting this option will provide you with the description for the code from the field above this one
Description A text description of a code.
Draft Permit Public Notification Date The date public notification was given.
Federal Grant Indicator Identifies a publicly owned waste treatment works (POTW) which obtained federal grant money.
Code Expansion for Federal Grant Indicator Selecting this option will provide you with the description for the code from the field above this one
Final Effluent Limits Indicator Indicates when the permittee has completed all necessary construction to achieve the ultimate effluent limitation in the permit.
Code Expansion for Final Effluent Limits Indicator Selecting this option will provide you with the description for the code from the field above this one
Flow Rate The average flow that a permitted facility was designed to accommodate.
Inactive Code Code which indicates an active facility.
Industrial Classification Identifies the facility's industrial classification.
Code Expansion for Industrial Classification Selecting this option will provide you with the description for the code from the field above this one
Latitude Latitude describing facility location.
Location City Name of the city or town in which the facility is located.
Location Name Name of the entity located at the facility's physical address.
Location Telephone Number Facility telephone number.
Location State State or territory in which the facility is located.
Code Expansion for Location State Selecting this option will provide you with the description for the code from the field above this one
Location Street 1 The first line of street information in the location address.
Location Street 2 The second line of street information in the location address.
Location ZIP Code Zip Code for the facility location address.
Longitude Longitude describing facility location.
Mailing City The city in the primary facility mailing address.
Mailing Name Mailing name used when standard labels ("ML" or "MN") are requested in a Generalized Retrieval, or DMR-keyed mailing labels are produced for a facility which has no alternate address on file.
Mailing State The state in the primary facility mailing address.
Mailing Street 1 The first line of street information in the primary facility mailing address.
Mailing Street 2 The second line of street information in the primary facility mailing address.
Mailing ZIP Code The zip code in the primary facility mailing address.
Major Discharge Indicator A one-character code designating that the facility is a major or minor discharger.
Method Code describing the procedure used to determine latitude and longitude coordinates.
Code Expansion for Method Selecting this option will provide you with the description for the code from the field above this one
Mileage Indicator A five-character field giving the length in miles of a particular facility stream segment.
Name 1 The first 30 characters of the facility name on the NPDES permit application.
Name 2 The second set of 30 characters of the facility name on the NPDES permit application.
NMP Final Schedule Indicates whether a final and enforceable Municipal Compliance Plan (MCP) schedule has been established to meet all statutory requirements.
Code Expansion for NMP Final Schedule Selecting this option will provide you with the description for the code from the field above this one
NMP Financial Status Code Indicates the financial fitness of the Publicly Owned Treatment Works (POTW).
Code Expansion for NMP Financial Status Code Selecting this option will provide you with the description for the code from the field above this one
NMP Fiscal Quarter Indicates the fiscal quarter during which the final enforceable municipal compliance plan (MCP) schedule is established.
Code Expansion for NMP Fiscal Quarter Selecting this option will provide you with the description for the code from the field above this one
Original Permit Issue Date The date the facility's first permit was issued.
Permit Expired Date The date the current permit will expire.
Permit Issued Date The date the current permit was issued/signed.
Pretreatment Code A one-character code which indicates a requirement to develop a pretreatment program.
Code Expansion for Pretreatment Code Selecting this option will provide you with the description for the code from the field above this one
Previous Year Automatic QNCR Status Indicates the status of reportable non-compliance as it appeared on the quarterly non-compliance report for the previous year.
Previous Year Manual QNCR Status Indicates the status of reportable non-compliance for major and minor facilities, as it appeared on the quarterly non-compliance report for the previous year.
Receiving Waters The name of the river, stream, tributary, lake, or other body of water into which the effluent is discharged.
Receiving Stream Classification Code A two-character field containing a code for the facility stream segment type.
Code Expansion for Receiving Stream Classification Code Selecting this option will provide you with the description for the code from the field above this one
River Basin A six-digit field used to identify the river basin in which the facility is located.
Code Expansion for River Basin Selecting this option will provide you with the description for the code from the field above this one
Scale Describes the scale used to determine the latitude and longitude coordinates.
Code Expansion for Scale Selecting this option will provide you with the description for the code from the field above this one
Stream Segment A four-character code to identify stretches of water from one significant event to another.
Type of Ownership A three-digit code describing ownership classification.
Code Expansion for Type of Ownership Selecting this option will provide you with the description for the code from the field above this one
Type of Permit Issued A one-character code indicating whether EPA or a state has issued the permit.
Code Expansion for Type of Permit Issued Selecting this option will provide you with the description for the code from the field above this one
USGS Water Basin Code A code to identify a facility's drainage water basins by their geographic location.
Code Expansion for USGS Water Basin Code Selecting this option will provide you with the description for the code from the field above this one
